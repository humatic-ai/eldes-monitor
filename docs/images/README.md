# Screenshots

This directory contains screenshots of the ELDES ESIM364 Monitor application.

## Adding Screenshots

To add a screenshot:

1. Take a screenshot of the application (e.g., temperature graph, device dashboard)
2. Save it in this directory with a descriptive filename (e.g., `temperature-graph.png`)
3. Update the main README.md to reference the screenshot
